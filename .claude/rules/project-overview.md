# TRAIVEL — Project Overview

TRAIVEL is an AI-powered travel planning web app for portfolio/CV. Wordmark: TR**AI**VEL ("AI" visually highlighted).

**3 core differentiators — no competitor currently has all three:**
1. Social trend signal layer — X/Twitter API, scored by volume + recency + engagement, sponsored content filtered out
2. Per-stop reason codes — structured typed chips: `social momentum`, `transport fit`, `food fit`, `budget fit`, `weather alternate ready`
3. Verification baked into generation — Foursquare + Open-Meteo called during Claude's tool_use, not as a post-processing step

**CV hook:**
> *"Không như lịch trình từ ChatGPT hay ảo giác thông tin, TRAIVEL xác minh từng điểm dừng với dữ liệu thực và cho bạn thấy **tại sao** mỗi điểm được chọn."*

**Constraints:** Solo build · $30/month budget · Goal is demo-able portfolio product with clean GitHub.
